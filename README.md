## Run Locally

Clone the project

```bash
  git clone https://github.com/houstonpearse/cube-web-app
```

Go to the project directory

```bash
  cd cube-web-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open http://localhost:5173/ in the browser
